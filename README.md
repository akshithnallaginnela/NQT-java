# Java NQT Practice

This repository contains a curated collection of essential Java practice programs. These scripts cover the core concepts, patterns, and fundamental logic building required to confidently solve coding questions commonly asked in technical exams like TCS NQT (National Qualifier Test) and other placement interviews. 

By mastering these basic standard problems, you will be well-prepared to solve at least one or more competitive programming challenges in your assessments.

## 📚 Problem Categories

### 1. Arrays
*   **`FindDuplicate.java`** - Identify duplicate elements efficiently.
*   **`MissingNumber.java`** - Find the missing number in a given continuous sequence.
*   **`NextGreaterElement.java`** - Find the next greater element for each array element (Stack concept).
*   **`PrintSubarrays.java`** - Generate and print all subarrays of an array.
*   **`PrintSubsets.java`** - Find all possible subsets (Backtracking concept).
*   **`RotateArray.java`** - Rotate an array by a given number of steps efficiently.
*   **`SecondLargestSmallest.java`** - Find the second largest and second smallest elements.
*   **`Sort012.java`** - Sort an array of 0s, 1s, and 2s in a single pass (Dutch National Flag algorithm).
*   **`TrappingRainWater.java`** - Calculate maximum rainwater that can be trapped between blocks.

### 2. Math & Number Theory
*   **`ArmstrongNumber.java`** - Check if a given number is an Armstrong number.
*   **`FibonacciSeries.java`** - Print or generate the optimal Fibonacci sequence.
*   **`GCD.java`** - Find the Greatest Common Divisor logically.
*   **`NumberSystemConversion.java`** - Convert numbers across different bases (Decimal, Binary, etc.).
*   **`PalindromeNumber.java`** - Determine whether an integer reads the same backward as forward.
*   **`PrimeNumber.java`** - Evaluate if a number is prime using optimized boundary checks.
*   **`ReverseNumber.java`** - Reverse the digits of a given number mathematically.

### 3. Strings
*   **`AnagramCheck.java`** - Analyze and check if two strings are valid anagrams.
*   **`CountVowelsConsonants.java`** - Parse strings to count alphabet categories.
*   **`LongestCommonSubstring.java`** - Find the longest contiguous substring shared between two strings.
*   **`RemoveDuplicates.java`** - Clean strings (or arrays) by stripping out redundant elements.

### 4. Data Structures
*   **`ArrayListOperations.java`** - Practice core, essential manipulation of dynamic arrays using Java's ArrayList.

## 🚀 Getting Started

To execute any of these files on your local machine:

1. Make sure you have the Java Development Kit (JDK) installed.
2. Open your terminal or command prompt.
3. Navigate to the repository folder.
4. Compile the specific Java file you want to test:
   ```bash
   javac FileName.java
   ```
5. Run the compiled Java class:
   ```bash
   java FileName
   ```

Happy Coding and Good Luck with your exams!
